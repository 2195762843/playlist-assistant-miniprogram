# 🎵 私人歌单整理助手

一个轻量级的微信小程序歌单管理工具，帮助您记录和整理音乐心情。纯本地存储，无需网络，安全私密。

## ✨ 功能特性

### 📋 歌单管理
- ✅ 新建歌单（名称、描述）
- ✅ 编辑歌单信息
- ✅ 删除歌单（含确认提示）
- ✅ 歌单置顶收藏
- ✅ 按创建时间排序

### 🎵 歌曲录入
- ✅ 歌曲名称（必填）
- ✅ 歌手信息（选填）
- ✅ 听歌心情备注（选填）
- ✅ 自动记录录入时间
- ✅ 歌曲收藏标记

### 🏷️ 标签分类
- ✅ 8 个预设标签（华语、欧美、日韩、粤语、摇滚、流行、古典、民谣）
- ✅ 自定义标签（最多 10 个）
- ✅ 标签颜色自定义
- ✅ 按标签筛选歌曲

### 🔍 搜索功能
- ✅ 歌单名称搜索
- ✅ 歌曲名称搜索
- ✅ 歌手搜索
- ✅ 心情备注搜索

### 📊 数据统计
- ✅ 歌单数量统计
- ✅ 歌曲总数统计
- ✅ 置顶歌单数量
- ✅ 收藏歌曲数量

###  数据安全
- ✅ 纯本地存储（wx.Storage）
- ✅ 关闭重开数据不丢失
- ✅ 无网络请求
- ✅ 无隐私数据收集

##  快速开始

### 环境要求
- 微信开发者工具 Stable 2.01+
- 基础库版本 2.16.0+
- 无需后端服务器

### 安装步骤

1. **克隆项目**
```bash
git clone https://github.com/2195762843/playlist-assistant-miniprogram.git
cd playlist-assistant-miniprogram
```

2. **导入项目**
   - 打开微信开发者工具
   - 选择「导入项目」
   - 选择项目目录
   - 填写 AppID（或使用测试号）

3. **编译运行**
   - 点击「编译」按钮
   - 在模拟器中预览效果
   - 扫码在真机调试

## 📝 使用说明

### 创建第一个歌单
1. 打开小程序，点击底部「+」按钮
2. 输入歌单名称（必填）
3. 填写描述（可选）
4. 选择标签（可选）
5. 点击「创建歌单」

### 添加歌曲
1. 进入歌单详情页
2. 点击底部「+」按钮
3. 填写歌曲信息
4. 选择标签（可选）
5. 点击「保存」

### 管理标签
1. 点击底部「标签」Tab
2. 查看预设标签
3. 点击「+ 添加自定义标签」
4. 输入名称、选择颜色
